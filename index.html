About the Author

Hello, my name is Jeff Henrichs and I am a Web Developer. I have graduated from Towson University, Class of 2019, with a Bachelors' Degree in Computer Science. At Towson, I have learned; programming languages like Java and JavaScript and query languages like SQL, technologies such as MySQL, VSCode, NodeJS and GitHub/GitLab, as well as concepts like OOP, Big O Notation and Version Control.

After college, I worked at Chiron Technologies Services, Inc. At Chiron, I have honed my HTML, CSS, and JavaScript skills by building and debugging web applications. I have sharpened my SQL skills by; creating, modifying, and retriving data from database tables and by protecting databases from SQL Injections, I also created the logic that connects the front-end to the backend with PHP and SQL. I have used technologies like PostgreSQL and Moodle and used frameworks like JQuery and Bootstrap.

Once my career at Chiron was over. I began to work for ValueMomentum Inc. Over at ValueMomentum, I continued to sharpen my knowledge on SQL and Java, while I learned about AWS and its services such as; S3 Volumes and EC2 Instances. I streamlined, debugged and organized data from AWS by creating automation scripts with Terraform and Python.
